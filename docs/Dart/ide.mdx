# Setup IDEs

You can code dart with any text editors which have support with dart extension/plugin/package.
These extension/plugin/package provide you with dart IntelliSense, syntax highlighting, 
run & debug support, and more.

<h3>Install Dart plugins</h3>

1. Open Android Studio.

2. Open **plugin preferences** (Configure > Plugins).

3. Select the Dart plugin and click **Install**.

4. Click **Restart** when prompted.

:::important

1. Open **plugin preferences** (on MacOS go to `Preferences > Plugins`; on Windows & Linux go to `File > Settings > Plugins`).

2. Go to **Marketplace** and search for the Dart plugin to install.
:::
