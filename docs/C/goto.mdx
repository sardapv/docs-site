### Goto

The `goto` statement allows us to transfer control of the program to the specified `label`.

```c
#include <stdio.h>

int main() {

  int i, inputs, noOfNumbers, number,sum = 0;
  double  average;
  printf("Enter no.of inputs :\n");
  scanf("%d", &inputs);
  for (i = 1; i <= inputs; ++i) {
    printf("%d. Enter a number: ", i);
    scanf("%d", &number);
    // go to jump if the user enters a negative number
    if (number < 0) {
      goto jump;
    }
    sum += number;
  }
  jump:
    average = sum / (i - 1);
    printf("Sum = %d\n", sum);
    printf("Average = %.2f", average);
  return 0;
}
```

**Output**

```
Enter no.of inputs :
4
1. Enter a number: 1
2. Enter a number: 2
3. Enter a number: 3
4. Enter a number: 3
Sum = 9
Average = 2.00
```
