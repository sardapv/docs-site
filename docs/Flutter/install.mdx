<h3>System Requirements :</h3>

<img height='50' src={useBaseUrl('/img/logos/windows.svg')}/>

**Windows 7/8/10** (64-bit), x86-64bit<br />
**Disk Space:** 1.5GB<br />
**Other Tools:** [PowerShell](https://docs.microsoft.com/en-us/powershell/scripting/windows-powershell/install/installing-windows-powershell?view=powershell-7.1), [Git for windows](https://git-scm.com/download/win), [Android Studio](https://developer.android.com/studio), [Visual Studio Code](https://code.visualstudio.com/download)(Optional), [JAVA 8](https://www.oracle.com/in/java/technologies/javase/javase-jdk8-downloads.html)

<h3> Steps to Install </h3>

Create a dir in `C:\` with the name of `src` and now open your command prompt(cmd) and head to `src` dir and clone the flutter repository from [Flutter GitHub repository](https://github.com/flutter/flutter/).

```shell
cd src\
git clone https://github.com/flutter/flutter.git -b stable --depth 1
```

<h3>Setup your flutter path</h3>

Now set the flutter path to your system environment. 

Press "**<img height='13' src={useBaseUrl('/img/logos/windows.svg')}/> + R**"
and run `SystemPropertiesAdvanced.exe` to open system properties

**<p align="center">OR</p>**

Press <img height='13' src={useBaseUrl('/img/logos/windows.svg')}/> key and search for `environment` and choose "**Edit the system environment variables**".

Click the "Environment Variables..." button in the bottom-right. In the user variables select `path` and click on **edit** to edit/manipulate the path.
If you see **Browse Directory...** and **Browse File** option then do this
  * In **Variable value**, continue with the existing paths this way `existing_paths;C:\src\flutter\bin\`.
  **<p align="center">Else</p>**
  * Add new **Variable value** with the path `C:\src\flutter\bin\`.

Click **ok** for all tabs that appear there and restart the system.

<h3>Checking the flutter command</h3>

Now check whether the **`flutter`** command is working or not.

```shell
flutter --version
```

If this gives you output, You are ready to flutter now. Else contact us [here](https://flutter.dev).

Now upgrade the flutter package you have cloned.

```shell
flutter upgrade
```
This will download the required packages.

<TabItem value="macos">

<h3>System Requirements :</h3>

<img height='50' src={useBaseUrl('/img/logos/apple.svg')}/>

**MacOS** (64-bit)<br />
**Disk Space:** 3GB<br />
**Other Tools:** [cocoapods](https://cocoapods.org/), [X-Code](https://apps.apple.com/in/app/xcode/id497799835?mt=12), [Android Studio](https://developer.android.com/studio), [Visual Studio Code](https://code.visualstudio.com/download)(Optional), [JAVA 8](https://www.oracle.com/in/java/technologies/javase/javase-jdk8-downloads.html)

<h3>Steps to Install</h3>

Create a dir with the name of `src` where ever you want in the system and now open your terminal(bash/zsh) and head to `src` dir and clone the flutter repository from [Flutter GitHub repository](https://github.com/flutter/flutter/).

```bash
git clone https://github.com/flutter/flutter.git -b stable --depth 1
```

<h3>Setup your flutter path</h3>

Now update the flutter path to your system environment.

Open your current terminal profile. If bash is your current/default terminal then open **`$HOME/.bash_profile`** or **`$HOME/.bashrc`**, If zsh is your current/default terminal then open **`$HOME/.zshrc`** .

After opening the file add the following line at bottom of the whole content

> **Note:** Update this **[Dir_path_where_src_exists]** with the path of your **src** directory.

```bash
export PATH="$PATH:[Dir_path_where_src_exists]/src/flutter/bin"
```

<h3>Checking the flutter command</h3>

Now check whether the **`flutter`** command is working or not.

```shell
flutter --version
```

If this gives you output, You are ready to flutter now. Else contact us [here](https://flutter.dev).

Now upgrade the flutter package you have cloned.

```shell
flutter upgrade
```
This will download the required packages.
